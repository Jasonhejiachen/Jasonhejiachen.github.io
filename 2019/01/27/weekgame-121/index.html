第一题：

这是原题的地址：https://leetcode-cn.com/contest/weekly-contest-121/problems/string-without-aaa-or-bbb/

我的思路：

用一个flaga flagb 判断a b 是否已经到了2个。然后再通过循环遍历将每一个字符输入进去。最后返回string

public String strWithout3a3b(int A, int B) {
    int longa =0;
    int longb =0;
    StringBuffer sb = new StringBuffer();

    while(A>0||B>0){
        if(A>B){
            if(longa<2){
                longa++;
                longb=0;
                sb.append('a');
                A--;
            }else{
                longa=0;
                sb.append('b');
                B--;
            }
        }else if(B>A){
            if(longb<2){
                longb++;
                longa=0;
                sb.append('b');
                B--;
            }else{
                longb=0;
                sb.append('a');
                A--;
            }
        }else if(A==B){
            if(longa<2){
                sb.append('a');
                sb.append('b');
            }else{
                sb.append('b');
                sb.append('a');
            }
            A--;B--;
        }
    }
    return sb.toString();
}
